CXP ReadMe

This file has a few links to CXP related resources.

CXP Protocol

The latest version of the CXP Protocol Specification is slightly out of date and will be upgraded in the next few days.

MedCommons CXP FireFox Client

The most recent version of the FireFox client is mccxp_0.9.21.xp. Click on this link to install in FireFox, then restart FireFox. If your browser security settings won't let you install from this link then right-click on the link to 'Save Link As' to the local file system. Once there simply load the installer using the 'Open File' menu.

Sample CXP SOAP client

The source for a trivial (and completely functional) CXP SOAP client is here. All other files needed to compile or run this program are available by downloading Apache Axis, Xerces 2.7.1 and other standard Java tools like javac and ant. Please contact Sean Doyle with any questions about this code or discuss it on the CCR Accel mailing list.

WSDL

The current WSDL for the CXP server is available on all gateway machines. This is derived from the WSDL provided by Roberto Ruggeri from Microsoft (Roberto.Ruggeri@microsoft.com).

Validation XSD

The current schema file used for CCR validation is CCR_20051107.xsd.
