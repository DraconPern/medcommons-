Mailing List directions

General information about the jakarta mailing lists can be found here. Please read these guidelines before subscribing or posting to the lists.

Cactus has 2 lists:

  • cactus-user@jakarta.apache.org: for users of Cactus
  • cactus-dev@jakarta.apache.org: for developers of Cactus

To subscribe to the user list, send an empty message to cactus-user-subscribe@jakarta.apache.org. Same for the dev mailing list.

To unsubscribe from the user list, send an empty message to cactus-user-unsubscribe@jakarta.apache.org. Same for the dev mailing list. Note that you must unsubscribe using the same email address you've used to subscribe. If you wish to unsubscribe from a different email address, use the following email address: cactus-user-unsubscribe-yourname=yourdomain@jakarta.apache.org (assuming the email address you wish to unsubscribe is yourname@yourdomain).

Once you're subscribed, send mail to cactus-user@jakarta.apache.org to post messages on the user list.

All emails sent to cactus-user@jakarta.apache.org are archived here.

All emails sent to cactus-dev@jakarta.apache.org are archived here.
