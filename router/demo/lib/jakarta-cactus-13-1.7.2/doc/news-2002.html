News and Events - Archives for 2002

28/12/2002 JavaDevWithAnt. Project demonstrating Ant, XDoclet, Struts, JUnit, Cactus, and Lucene.
01/12/2002 Added automatic script support for JBoss 3.x. They are delivered as part of the Cactus distribution.
07/11/2002 Release 1.0 of the Cactus plugin for Maven is out. This is certainly the easiest way to run Cactus tests so far. This plugin is part of the core Maven plugins and is thus delivered with Maven.
13/09/2002 New tutorial that explains how to build Cactus from the sources. Check the Build Howto now !
31/08/2002 Cactus 1.4.1 has been released ! It is purely a bug fix release. Get it from the Download area. Complete change log is available here.
25/08/2002 Cactus 1.4 final is released ! Get it from the Download area. Complete change log is available here.
29/07/2002 Cactus 1.4 beta 1 is released ! Get it from the Download area. Complete change log is available here.
22/07/2002 Added automatic script support for Orion 1.6. They are delivered as part of the Cactus distribution.
28/06/2002 Article from Sheldon Wosnick explaining how to use Cactus inside WebSphere Studio Application Developer (WSAD).
19/06/2002 Added automatic script support for WebLogic 7.0. They are delivered as part of the Cactus distribution.
09/06/2002 Added Test Coverage Reports that shows how good (or how bad !) Cactus self tests are. We have used Clover to generate these reports.
19/05/2002 New Test flexibly with AspectJ and mock objects article by Nick Lesiecki: It demonstrates how to use AspectJ in association with Mock Objects to provide greater test flexibility. It also shows how AspectJ can complement Cactus tests.
18/05/2002 Kaarle Kaila has started KaCoMa: KaCoMa (abbreviation for Kaarle's ContentManagement until further notice) is a ContentManagement system for web-sites that serves as a tutorial for beginners of custom taglibraries and/or Unit testing of corresponding software.
14/05/2002 New Quick start tutorial that explains how to run Cactus tests quickly in Tomcat.
13/05/2002 New tutorial that explains a new of running Cactus tests: through a browser. Check the TestRunner Howto now !
21/04/2002 Cactus 1.3 is released ! Get it from the Download area. Complete change log is available here.
21/04/2002 New Tested On ... page which lists what containers have been tested with Cactus.
20/04/2002 New Cactus users page.
20/04/2002 New logo proposals by Alessandro Ceglia, Dimenzio Design Works and Arthur Jasso.
20/04/2002 New Security tutorial.
08/04/2002 Cactus BOF at OT2002 led by Vincent Massol from OCTO Technology.
25/03/2002 Speech on Cactus and the Jakarta Community entitled "Itineraire d'un logiciel libre: Jakarta Cactus" at the Luxembourg Free Open Source days ("Journees du Libre"). Led by Vincent Massol from OCTO Technology.
21/03/2002 Cactus is now integrated in Objecteering. Read here for details about this integration.
27/02/2002 Together Control Center 6.0 now incorporates the concept of Test Plan and has tight integration with JUnit, HttpUnit and Cactus.
22/02/2002 New logo proposals by Alessandro Ceglia !
10/02/2002 New Web site Statistics page.
20/01/2002 Link to Cactus nightly build result added to the left menu in the developer section.
19/01/2002 New JBuilder 5 tutorial for Cactus integration. Thanks Sean !
