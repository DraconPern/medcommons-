    Nightly Build Results

    Every day, the main Cactus build is run, the cactus jars are generated, the full suite of tests is executed and a nightly cactus distribution is packaged. This is possible thanks to GUMP.

    Please find below the links to the Cactus Nightly Build results:

    All Gump results are available here.

    If the nightly build is successful, Gump posts the packaged Cactus distributable in the Cactus nightly download area.
