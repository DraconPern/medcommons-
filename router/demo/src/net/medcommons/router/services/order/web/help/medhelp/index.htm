MedCommons Viewer Help

The MedCommons Viewer allows you to view CCR's as well as attached images and documents.

The MedCommons Viewer is split into six distinct sections (click on a section to view instructions):

  • Thumbnail Navigator
  • The Thumbnail Navigator contains clickable thumbnails of a CCR and all of its related attachments including image series & PDF's.
  • Command Buttons
  • The command buttons control image viewing functions.
  • Toolbar
  • The toolbar contains the Help and Print functions.
  • Series List
  • The series list is for navigation of the CCR and its components.
  • Series Dropdown
  • The series dropdown is for navigation of the CCR and its components.
  • Viewing Area
  • The viewing area displays the currently selected component of the CCR.



Thumbnail Navigator

The thumbnail navigator allows you to easily view all of the attached components to a CCR. these may include PDF's, images, and the body of the CCR itself. When a particular component is selected, it is highlighted with a white outline.

Command Buttons

The command buttons are functional only when a series of images is selected. There are three command buttons in total:

  1. Zoom Button
  2. The zoom tool is the default tool selected when you click to an image series. You can zoom in on an image with the zoom tool by left clicking, dragging a square around a section of the image you would like magnified, and letting go of the mouse button.
  3. Reset Button
  4. The reset button resets an image to its original overlay and magnification setting.
  5. DICOM Overlay Button
  6. The DICOM Overlay displays DICOM information for image series. It has three states - off, simple, and complete.




Mouse Scroll Wheel Rapid Index Function

If you are viewing an image series, rotating your mouse's scroll wheel will cause a 3-D image to rotate, and a CT series to move up and down through its slices (see below).

Toolbar

The MedCommons Toolbar currently allows access to the help and print functions. To move the MedCommons toolbar, you can click on the top area of the window and drag it around your browser window.

Series List

The series list allows for quick navigation of a CCR's components.

Series Dropdown

The Series Dropdown menu provides a more detailed listing of CCR Components than the Series List, and allows for quick navigation of a CCR's components.



^back to top^




Viewing Area

The viewing area contains whichever component of a CCR is currently selected for viewing. If the currently selected component is a PDF, the viewing area also contains standard PDF controls, like "print" and "save to disk".

Rapid Image to Index function

When you open an image series in the viewing area, and would like to index through a large series quickly, try placing the mouse between the Thumbnail Navigator and the viewing area, until it looks like a double ended arrow pointing north and south. If you left click and drag up and down in this area, the selected image will index quickly through its stack.
