About DICOM Data Liberator (DDL)

The DDL (Dicom Data Liberator) is a client application which is used to import DICOM data into the MedCommons system or to export data from MedCommons.

Data is imported by into the DDL via DICOM CSTORE from a third party DICOM device or from the Upload DICOM Folder menu item.

Data is exported in one of two ways depending on the ExportMethod configuration. If this value is set to CSTORE then the results are sent to the DICOM target defined by the dicomRemoteAEHost, dicomRemoteAEPort, dicomRemoteAeTitle values. If the value is set to FILE - the files are placed into the directory specified by exportDirecory.
