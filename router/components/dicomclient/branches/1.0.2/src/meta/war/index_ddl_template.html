This is the installer page for the DICOM Data Liberator (DDL). The DDL is a Java Web Start application which is installed on your local client machine. Currently this installer runs best under Microsoft Windows and Linux. Other operating systems (MacOSX) will be fully supported in future releases.

A notification window is displayed when DDL has completed startup.

Instructions If Java is not already installed on the client machine, it will automatically download and start the install of Java 6, and then after the install, it will launch the DDL Application automatically.

If you are prompted with a certificate challenge - select 'trust' to trust Sun (for Java) and MedCommons (for DDL).

If your browser can not detect that Java6 Java Web Start has been installed (this may be the case in MacOSX) you may directly invoke the DDL here. Depending on your browser configuration this may launch the DDL or just download the ddl.jnlp file to your browser's download directory. If this is the case then open the download folder and then open this file.
