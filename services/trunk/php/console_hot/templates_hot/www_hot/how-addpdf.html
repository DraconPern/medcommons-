How To: Add PDF Documents to DICOM

last updated on Sep 9,2010

         
    Click Edit on the HealthURL  
         
    Select Add PDF Document  
         
    Browse to the file  
         
    Verify and click the red !
The verification step is designed to reduce the risk of patient mixup. If you attached the wrong document, click Edit on the CCR File menu and click the X to delete the document in red.
