  Sample Radiology CD
You can try out MedCommons On Demand even if you don't have a radiology CD handy. Before you start the Upload Form, download a sample DICOM folder (Windows) or CD (Mac) and use it as if you had a CD already in the drive.		  
 

For Windows:

      
         
    Download a DICOM demo folder (3MB).
Proceed as if it's a CD.
Un-zip the file to create a folder on your desktop.
Proceed as with a real CD but navigate to the Desktop...		  
         
    Select the folder
Click "open" 		 
         
 

For Mac:

      
         
    Download a sample CD image (3MB).
Proceed as with a CD.
