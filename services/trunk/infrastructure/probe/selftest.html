Self-Test My Browser Connection To MedCommons

This page is for customer use. One purpose is to assist the MedCommons Help Desk to diagnose particular problems with a customer's MedCommons experience.

All of these tests put up a little display and recyle various performance numbers as each test is performed

Loopback Test

This in and out of the MedCommons appliance as quickly as possible. Look at "Round Trip Time" to get a feel for the responsiveness of the system.

  • Basic Loopback

Upload Tests

Some of these tests can take a long time. To get a very precise measure, run the "Upload Nothing" test first, and subtract its round-trip time from the specific upload test result to get the actual time to move the data upstream.

  • Upload Nothing
  • Upload 1K Bytes
  • Upload 128K Bytes (one minute cycle)
  • Upload 1M Bytes (one minute cycle) very slow!!!

Download Tests

This test will download a 10M document to your system directly from the MedCommons Gateway that services your account. Detailed instructions after the link

  • Download 10M Bytes (two minute cycle) very slow!!!!
