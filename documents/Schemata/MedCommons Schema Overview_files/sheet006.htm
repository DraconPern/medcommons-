Order
Patient string() Points to the Patient Record inside central
Recipe * string() Points to Recipe Records Specifying Execution Steps
PaymentRequest string() Details as sent to credit card processor
PaymentResponse string() Response from Credit Card Processing Abank
TrackingNumber string() Only filled in for a completed order
Template string() The URL of Forms Template used to submit this order
SignedPDF string() The URL of Final Closed Signed PDF document used to submit this order
